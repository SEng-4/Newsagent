## Project Setup

1. Clone the repository using GitHub Desktop.
2. Install the following extensions in Visual Studio Code:
 * [Extension Pack for Java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack)
 * [Spring Boot Extension Pack](https://marketplace.visualstudio.com/items?itemName=vmware.vscode-boot-dev-pack)
3. Open the `Newsagent` folder in VS Code.
4. Click the Spring Boot Dashboard icon in the left-hand sidebar.
5. To run the server, click the "Run" icon next to `newsagent` in the Apps section of the Spring Boot Dashboard.
